Häufig gestellte Fragen

Werden meine Daten auf einen Server hochgeladen?

Nein. Das Scannen findet lokal in deinem Browser mit JavaScript statt. Es werden niemals Dateien oder Daten auf unsere Server hochgeladen.

Welche Dateitypen kann ich scannen?

Du kannst .jar-Dateien, .zip-Archive, gesamte Minecraft Client-Ordner und einzelne Mod-Dateien scannen.

Wie genau ist die Erkennung?

Unser Scanner verwendet Pattern-Matching gegen eine umfassende Datenbank bekannter Hacked Clients. Obwohl sehr genau, ist kein System 100% perfekt.

Können falsche Erkennungen auftreten?

Ja, einige legitime Mods können Warnungen auslösen. Überprüfe immer den Kontext und die Risikostufe jeder Erkennung.
